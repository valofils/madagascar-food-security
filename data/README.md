# Data Sources

## Raw
- `hdx_ipc_mdg.json` — IPC phase data from HDX HAPI (Madagascar, 2017–present)
- `fewsnet_mdg.csv` — FEWS NET classifications from FDW API

## External
- Climate/rainfall indicators (CHIRPS via FEWS NET)
- Population estimates

## Schema
Each record: district × period × IPC phase (1–5) + population estimates
